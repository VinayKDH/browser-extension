üîß Manual Testing

Perform these manual tests to verify extension behavior:

1. Extension Installation

  • Check if extension icon appears in browser toolbar
  • Verify extension is enabled in browser extensions page
  • Confirm no error messages in browser console

2. Popup Functionality

  • Click extension icon to open popup
  • Type text in search input and press Enter
  • Click "Go" button
  • Verify new tab opens with Tens AI

3. Content Script Injection

  • Look for floating "Tens AI" button on this page
  • Click the floating button
  • Verify it opens Tens AI in new tab

4. Keyboard Shortcuts

  • Press Alt+Shift+T on this page
  • Verify Tens AI opens in new tab
  • Select some text and press Alt+Shift+T
  • Verify selected text is passed to Tens AI

5. Context Menus

  • Right-click on extension icon (if supported)
  • Select "Open Tens AI" from context menu
  • Select text on page, right-click and choose "Search in Tens AI"

6. Tens AI Connection

  • Verify extension can open dev2.tens-ai.com
  • Check if search queries are properly passed
  • Test with various text inputs
